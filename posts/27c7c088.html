个人Java开发环境搭建（闲时更新）

2020-08-03 | 笔记 | | 阅读数
字数统计: 748

前言

本文主要用于个人记录开发环境的搭建，便于环境迁移时的重建工作，文中提及相关文件可在曲奇云盘中获取

Java

  1. Windows
    运行jdk-8u181-windows-x64.exe以安装JDK
  2. Linux
    解压tar -zxvf jdk-xxx.tar.gz，配置环境变量

Maven

解压apache-maven-3.6.3-bin.zip，配置环境变量

Git

  1. 安装Git-2.27.0-64-bit.exe
  2. 配置ssh key 进入.ssh文件夹下复制公钥id_rsa_pub文本内容，在Github->settings->ssh and gpg keys->new ssh key 多用户Git配置请参考本地配置Git多账户

Mysql

  1. 解压mysql-8.0.20-winx64.zip，进入bin目录
  2. 创建my.ini
  3. 初始化Mysql
    在命令行窗口进入bin目录，执行mysqld --initialize --console，记录随机生成的密码
  4. 安装并启动
  5. 设置密码
    mysql -u root -p输入密码进入mysql控制台，通过set password = password('xxx');修改默认密码
  6. 忘记默认密码？

NPM

  1. nvm：用于npm版本控制，运行nvm-setup.exe
  2. 设置nvm镜像：
  3. 安装指定版本nodejs
  4. 设置npm镜像
  5. 使用cnpm

博客迁移

有以下三种方法

  1. Cloud Studio线上开发
  2. 从Coding仓库克隆到本地后安装hexo和其他依赖即可
  3. 复制scaffolds、source、themes、_config.yml、package.json，在新文件夹下安装hexo和其他依赖

虚拟机相关

Vmware

  1. 解压vmware-pro15.zip，安装
  2. 使用镜像CentOS-7-x86_64-Minimal-1908.iso，创建虚拟机
